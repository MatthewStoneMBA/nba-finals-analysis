# NBA Finals Data Analysis Project

An R-based data analysis project examining NBA Finals historical data. Explores team and player performance trends across championship games.

## What It Does

- Analyzes NBA Finals game and player data
- Identifies patterns in championship-winning teams
- Visualizes performance metrics and trends

## Stack

- R
- R Markdown

## Files

- `NBA Finals Data Analysis Project/NBA Finals Data Analysis Project.Rmd` — main analysis
- `NBA Finals Data Analysis Project/championsdata.csv` — dataset
