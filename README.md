# NBA Finals: When Stats Matter

An R-based data analysis project examining NBA Finals historical data to answer two research questions:

1. **What key statistics most impact wins?**
2. **What statistics have been changing throughout the years?**

## What It Does

- Loads and cleans NBA Finals champions dataset
- Performs exploratory data analysis on team performance stats
- Identifies which statistics (points, rebounds, assists, etc.) correlate most with winning
- Tracks how the game has evolved statistically over time
- Uses regression modeling and correlation analysis to surface key drivers

## Stack

- R / R Markdown
- tidyverse, ggplot2, moderndive, GGally, broom, skimr, wesanderson

## Files

- `NBA Finals Data Analysis Project/NBA Finals Data Analysis Project.Rmd` — full analysis
